About Visual Studio Code

Visual Studio Code is an integrated development environment(IDE) used to program websites, web apps etc. To set it up first u have to download it. Then when the donload is done, there is a search bar on the screen and on it type "live server" and install it to live preview your website. After that make a folder on your desktop. Then on visual studio, open this folder and make a file to start coding.


About Console Screen

Console screen is used to check whether your variables and calculations done in the website are correct and it is very simple to open it. On the website, just right click and click "Inspect". Then on the top tab navigator click "Console to view the console screen.
